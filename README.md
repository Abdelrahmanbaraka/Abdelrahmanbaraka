# Java Calculator App

## Overview
This project is a simple console-based calculator application built in Java.  
It allows the user to enter two numbers, choose a mathematical operation, and displays the result.

---

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Invalid operation handling

---

## Technologies Used
- Java
- Scanner class for user input

---

## How It Works
1. The program asks the user to enter the first number.
2. The program asks for the second number.
3. A menu of operations is displayed:
   - Addition
   - Subtraction
   - Multiplication
   - Division
4. The user selects an operation by entering its number.
5. The program performs the calculation and prints the result.

---

## Example
```text
Enter the first number: 10
Enter the second number: 5
Select an operation:
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
Enter the operation number: 4
Result: 2.0
